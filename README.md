# This is the vimrc file of Rei, do not use this if you are not rei

## Installation instructions for vim
```(bash)
mkdir -p ~/.vim/pack/{anyname}/start

```

cd into it
```(bash)
cd ~/.vim/pack/{anyname}/start
```

then git clone repo
```(bash)
git clone <repo>
```

load helptags
```(bash)
vim -u NONE -c "helptags {plugindir}/doc" -c q

```
