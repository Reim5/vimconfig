# Rei's vimrc

## Installation instructions for vim
```(bash)
mkdir -p ~/.vim/pack/{anyname}/start
```

cd into it
```(bash)
cd ~/.vim/pack/{anyname}/start
```

then git clone repo
```(bash)
git clone <repo>
```

load helptags
```(bash)
vim -u NONE -c "helptags {plugindir}/doc" -c q

```
